프로젝트 제목: Element Puzzle Drag 

게임 컨셉: 하나의 드롭을 드래그로 이동시키며 퍼즐 구조를 재배치하고, 연쇄 콤보를 통해 공격을 수행하는 실시간 퍼즐 게임

개발 범위:
퍼즐 시스템:
  6x5 그리드 구성
  드롭 종류 6종 (불, 물, 나무, 빛, 어둠, 회복)
  드래그 기반 경로 이동 구현 (최대 10초)
매칭 및 연쇄 시스템
  3개 이상 매칭 판정
  낙하 및 연쇄 콤보 처리
전투 시스템
  속성별 공격 계산 (5속성 + 회복)
  타겟 포커스 기능
스킬 시스템
  플레이어 스킬 2종 구현
  ㄴ특정 속성 공격력 증가
  ㄴ드롭 변환
스테이지 구성
  총 10개 스테이지
  스테이지별 몬스터 체력 및 공격 주기 설정
UI 시스템
  전투 UI
  체력바
  게임 시작 / 클리어 / 게임오버 화면

퍼즐 조작 전:
  화면에 보이는 몬스터를 터치하여 조작 후 공격할 몬스터를 포커스할 수 있다. 포커스된 몬스터를 다시 터치하면 포커스가 풀린다.
  플레이어 스킬을 사용해 특정 속성의 공격력을 한 턴 높이거나, 퍼즐에 배치된 드롭 일부를 다른 드롭으로 바꿀 수 있다.
퍼즐 조작:
<img width="3209" height="1170" alt="퍼즐 조작 예시" src="https://github.com/user-attachments/assets/2295b33d-3944-47b5-98ca-24298ae0c466" />
  퍼즐판에서 하나의 드롭을 터치하고 움직이면 조작 단계가 된다.
  드롭을 드래그해서 이동시키면 이동된 자리에 있던 드롭과 자리가 바뀐다.
  조작은 최대 10초 가능하고, 조작 10초가 지나거나, 그 전에 드롭을 드래그하던 손가락을 놓으면 조작이 종료된다.
퍼즐 조작 후 연쇄:
  <img width="3209" height="2202" alt="퍼즐 연쇄 예시" src="https://github.com/user-attachments/assets/8d0ca89b-a105-4700-b8ba-24f3223f22a7" />
  퍼즐 조작이 끝나면, 퍼즐에서 가로나 세로로 3개 이상 연결된 드롭이 있으면 그 줄로 연결된 드롭이 전부 사라진다.
  드롭이 사라지면 그 위에 있던 드롭은 아래로 떨어지고, 빈 공간은 새로운 랜덤 드롭이 떨어진다.
  떨어진 드롭들로 또 가로나 세로로 3개 이상 연결된 드롭이 있으면 드롭이 사라지고, 3개 이상 연결된 드롭이 없어질 때까지 반복한다.
플레이어 공격 턴:
  연쇄를 포함한 조작이 전부 끝나면 지금까지 지워진 드롭들을 통해 몬스터에 줄 공격량을 계산한다.
  이전 조작에서 지워졌던 드롭의 속성이 있다면 해당 속성은 공격을 진행한다. 이때, 분홍색 드롭은 공격이 아닌 체력을 회복시킨다.
  포커스 된 몬스터가 있다면 모든 속성 공격이 그 몬스터를 향하고, 몬스터가 포커스되어있지 않다면 각 속성의 공격이 랜덤한 몬스터를 향하게 된다. 
몬스터 공격 턴:
  플레이어 공격 후 살아남은 몬스터가 있으면 턴 수가 1 줄어들고, 턴 수가 0이 되면 플레이어를 공격하고 공격 턴 수가 올라간다.
스테이지 클리어:
  화면에 보이는 몬스터가 전부 죽으면 다음 스테이지로 넘어간다. 총 10개의 스테이지를 전부 클리어하면 게임이 클리어된다.
게임 오버:
  플레이어의 체력이 0이 되면 게임에서 패배한다.

개발 일정:
1주차: 리소스 확보, 기본 퍼즐 화면 구현
2주차: 퍼즐 시스템 구현(드롭 드래그)
3주차: 퍼즐 시스템 구현(연쇄)
4주차: 플레이어 스킬 구현, 플레이어 공격 구현
5주차: 몬스터 공격 구현
6주차: 스테이지 구현 및 레벨 밸런싱
7주차: 시작 화면, 게임 클리 화면, 게임 오버 화면 구현
8주차: 게임 루프 구현 및 디버깅, 밸런스 테스트
