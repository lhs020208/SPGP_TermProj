# 프로젝트 제목: Element Puzzle Drag


# 게임 컨셉

> 하나의 드롭을 드래그로 이동시키며 퍼즐 구조를 재배치하고, 연쇄 콤보를 통해 공격을 수행하는 실시간 퍼즐 게임

# 개발 범위

## 1. 퍼즐 시스템
- 6 x 5 그리드 구성
- 드롭 종류 6종 (불, 물, 나무, 빛, 어둠, 회복)
- 드래그 기반 경로 이동 구현 (최대 10초)

## 2. 매칭 및 연쇄 시스템
- 3개 이상 매칭 판정
- 드롭 제거 후 낙하 처리
- 연쇄 콤보 발생 및 반복 처리

## 3. 전투 시스템
- 속성별 공격 계산 (5속성 + 회복)
- 몬스터 타겟 포커스 기능

## 4. 스킬 시스템
- 플레이어 스킬 2종
  - 특정 속성 공격력 증가
  - 드롭 변환

## 5. 스테이지 구성
- 총 10개 스테이지
- 스테이지별 몬스터 체력 및 공격 주기 설정

## 6. UI 시스템
- 전투 UI
- 플레이어 체력바
- 게임 시작 / 클리어 / 게임 오버 화면

- 몬스터 턴 수 감소
- 턴이 0이 되면 플레이어 공격


## 6. 결과

### 스테이지 클리어
- 모든 몬스터 처치 시 다음 스테이지 이동
- 총 10 스테이지 클리어 시 게임 종료

### 게임 오버
- 플레이어 체력이 0이 되면 패배
